# Social Media App

## Getting Started

1. Clone this repository:
   ```
   git clone https://github.com/MrAlob/social-media-app.git
   cd social-media-app
   ```

2. Install dependencies:
   ```
   npm install
   ```

3. Start the development server (if using Vite or similar):
   ```
   npm run dev
   ```
   Or open `index.html` directly in your browser for a static build.
